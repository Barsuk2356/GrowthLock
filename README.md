# GrowthLock

AI-платформа саморазвития: цели, дневные задачи, материалы, AI-тесты, AI-наставник и очки навыков. Поддерживает AI-провайдеры OpenRouter, DeepSeek, Gemini, Zveno, универсальные OpenAI-compatible API и локальную Ollama.

> Для запуска на хостинге смотри `LAUNCH_CHECKLIST.md`.


## Как запустить сайт с AI

1. Установи зависимости:

```bash
npm install
```

2. Создай файл `.env` рядом с `server.js`:

```bash
cp .env.example .env
```

3. Выбери провайдера.

### DeepSeek

```env
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=your_deepseek_api_key_here
DEEPSEEK_MODEL=deepseek-chat
PORT=3000
```

### OpenRouter

```env
AI_PROVIDER=openrouter
OPENROUTER_API_KEY=your_openrouter_api_key_here
OPENROUTER_MODEL=openai/gpt-oss-20b:free
PORT=3000
```

4. Запусти:

```bash
npm start
```

5. Открой:

```text
http://localhost:3000
```

## Что подключено

- `/api/ai/status` — показывает, какой AI-провайдер выбран.
- `/api/ai/goal-review` — AI анализирует цель, оценивает её по 100-балльной шкале, предлагает улучшенную формулировку, первые шаги и материалы для изучения. После оценки цель сохраняется в раздел «Цели» вместе с материалами; задачи можно привязывать к конкретной цели. Ссылки на материалы подтягиваются из реальных открытых источников, например Open Library и Wikipedia, а не просто через поисковую строку.
- `/api/ai/quiz` — AI анализирует текст задачи или материала и генерирует 3 вопроса на понимание с учётом выбранной сложности: лёгкая, средняя или сложная.
- `/api/ai/check-quiz` — AI проверяет ответы по скрытым критериям и решает, засчитывать ли задачу или материал. У лёгкого теста проверка мягче, у среднего вопросы формулируются простыми словами, у сложного — строже и полезнее для глубокого усвоения. После успешного теста AI сам начисляет очки любых навыков: интеллект, психология, финансы, сила, выносливость и т.д.
- `/api/ai/explain-mistakes` — AI-наставник объясняет ошибки после проваленного теста, даёт план повторения, примеры более сильных ответов и реальные материалы для повторного изучения.
- `terms.html` — пользовательское соглашение.
- `privacy.html` — политика конфиденциальности.

## Дополнительно

- Добавлена регистрация/вход через backend: у каждого пользователя свой прогресс, цели, задачи, материалы и очки навыков. Данные сохраняются на сервере в JSON-базу `.data/growthlock-db.json`, а в браузере остаётся локальный кэш. Поэтому при обновлении `index.html`/`server.js` прогресс пользователей не должен стираться, если не удалять `.data`.
- В разделе цели можно переключаться между целями, удалять цель, видеть прогресс и материалы конкретной цели.
- AI предлагает стартовые задачи: их можно добавить по одной или сразу все к выбранной цели.
- Есть экспорт/импорт прогресса в JSON, чтобы перенести данные или сделать резервную копию.
- В интерфейсе есть кнопка «Сбросить прогресс», если нужно начать заново.

Важно: ключи хранятся только в `.env`, не в HTML. Файл `.env` добавлен в `.gitignore`.

## Важно про обновления

При локальном запуске пользовательские данные лежат в:

```text
.data/growthlock-db.json
```

Если обновляешь сайт, можно заменять файлы кода:

```text
index.html
server.js
package.json
README.md
```

Но **не удаляй папку `.data`**, иначе серверная база пользователей исчезнет. Для настоящего публичного сайта лучше вынести данные во внешнюю БД: PostgreSQL, Supabase или Firebase.
