# GrowthLock — прототип

## Как запустить сайт с AI

1. Установи зависимости:

```bash
npm install
```

2. Создай файл `.env` рядом с `server.js`:

```bash
cp .env.example .env
```

3. Выбери провайдера.

### DeepSeek

```env
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=sk-...
DEEPSEEK_MODEL=deepseek-chat
PORT=3000
```

### OpenRouter

```env
AI_PROVIDER=openrouter
OPENROUTER_API_KEY=sk-or-v1-...
OPENROUTER_MODEL=openai/gpt-oss-20b:free
PORT=3000
```

4. Запусти:

```bash
npm start
```

5. Открой:

```text
http://localhost:3000
```

## Что подключено

- `/api/ai/status` — показывает, какой AI-провайдер выбран.
- `/api/ai/goal-review` — AI анализирует цель, оценивает её по 100-балльной шкале, предлагает улучшенную формулировку, первые шаги и материалы для изучения. После оценки цель сохраняется в раздел «Цели» вместе с материалами; задачи можно привязывать к конкретной цели. Ссылки на материалы подтягиваются из реальных открытых источников, например Open Library и Wikipedia, а не просто через поисковую строку.
- `/api/ai/quiz` — AI анализирует текст задачи или материала и генерирует 3 вопроса на понимание с учётом выбранной сложности: лёгкая, средняя или сложная.
- `/api/ai/check-quiz` — AI проверяет ответы по скрытым критериям и решает, засчитывать ли задачу или материал. У лёгкого теста проверка мягче, у среднего вопросы формулируются простыми словами, у сложного — строже и полезнее для глубокого усвоения. После успешного теста AI сам начисляет очки любых навыков: интеллект, психология, финансы, сила, выносливость и т.д.
- `/api/ai/explain-mistakes` — AI-наставник объясняет ошибки после проваленного теста, даёт план повторения, примеры более сильных ответов и реальные материалы для повторного изучения.

Важно: ключи хранятся только в `.env`, не в HTML. Файл `.env` добавлен в `.gitignore`.
