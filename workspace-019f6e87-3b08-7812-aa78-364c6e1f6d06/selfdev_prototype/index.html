Стартовый режим

Сейчас аккаунт чистый: нет задач, нет провалов и нет блокировок. Ограничения включатся только после невыполненных задач или проваленных AI-тестов.

Логика будущего сайта

Так будет работать продукт после подключения API и базы данных.

1. Пользователь ставит цель

AI оценивает понятность, срок, реалистичность и предлагает улучшения.

2. AI подбирает материалы

Книги, видео, статьи и задания под конкретную цель пользователя.

3. Пользователь выполняет задачи

Для каждой задачи AI генерирует короткий тест из 3 вопросов по её смыслу.

4. Нарушения блокируют функции

Пока задача не подтверждена, часть сайта закрывается — мотивация не сливаться.
